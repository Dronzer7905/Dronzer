# Dronzer Roadmap (v2.x)

Following the successful release of **v2.0.0**, the core team is focused on expanding the Gateway's capabilities for the remainder of the year:

## Q3 (Current)
- **Advanced Load Balancing**: Introducing Round-Robin and Latency-Based routing strategies in addition to the current Task-Aware Engine.
- **Custom Provider Support**: Allowing users to define entirely custom API endpoints and models in the UI without database migrations.

## Q4
- **Edge Deployment Proxies**: Lightweight Rust-based edge nodes for deploying the Dronzer router directly to Cloudflare Workers or Vercel Edge.
- **Cost Analytics Dashboard**: Beautiful visualizations breaking down token usage and cost-savings across all providers.
